## 📊 系统概览

### 核心功能
- ✅ 会员管理（会员卡、课时包、积分、等级）
- ✅ 课程管理（课程模板、排课、教练管理）
- ✅ 预约服务（预约、取消、签到、提醒）
- ✅ 支付中心（微信支付、订单管理、退款）
- ✅ 消息通知（微信模板消息、短信）
- ✅ 数据统计（业务报表、运营看板）

### 技术栈
- **前端:** 微信小程序 + React 管理后台
- **后端:** Node.js (NestJS)
- **数据库:** PostgreSQL 14+
- **缓存:** Redis 7+
- **部署:** Docker + Kubernetes
